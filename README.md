## Установка и запуск
1. Клонируйте репозиторий:

   ```bash git clone https://github.com/MMadiyar417/star-wars```
2. Перейти в директорию проекта:  cd dashboard
3. Установить зависимости:
    npm install
4.Запустить проект:
  npm start
## Авторизация:
   - **Данные авторизации:**
     - Логин: `admin`
